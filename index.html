Gradient Grid

Description

The Gradient Grid is a generative web application that allows the user to create a personalized pattern for their use. Upon opening, the squares will be given a random fill color; refresh the page to change the color. Use the Graphical User Interface on the right to change the parameters listed below.

Parameters:

  • description - show/hide the description
  • size - changes the size of the squares equally
  • squares - the amount of squares shown in each corner
  • stroke - changes the weight of the stroke
  • nostroke - enable/disable the stroke
  • strokecolor - changes the color of the square's stroke
  • colorfill - fills the squares with a random color
  • backcolor - changes the background color
